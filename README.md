# FitGen 💪

Generador de rutinas de ejercicio personalizadas con IA.

## ¿Qué hace?

Completás un formulario con tus datos (edad, peso, objetivo, disponibilidad), y FitGen genera una rutina personalizada usando Groq AI y te la envía por email.

## Stack

- **Backend**: ASP.NET Core Web API
- **Arquitectura**: Clean Architecture (Domain, Application, Infrastructure, API)
- **IA**: Groq
- **Email**: Gmail SMTP
- **Tests**: xUnit + Moq

## Características

✅ Formulario web responsivo
✅ Validaciones en frontend y backend
✅ Integración con IA (Groq)
✅ Envío de rutinas por email en HTML
✅ Manejo de errores con middleware
✅ Unit tests con Moq

## Tecnologías

- .NET 8
- ASP.NET Core
- MailKit
- Moq
- xUnit
